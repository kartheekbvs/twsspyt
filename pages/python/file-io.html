📘 3.1 Decision Making: The Fork on the Road The statement

if cond: this else: that where cond is a Boolean expression, i.e., has the value of True or False, will execute the code block this if the cond value is True and that otherwise. That means, in either case, one of the statements this or that will be performed (but not both). So this gives us the ability to make a decision about which part of the code one wants to perform.
