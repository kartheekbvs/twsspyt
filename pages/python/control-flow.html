Control Flow

Control flow statements determine the order in which code executes. Python provides if/elif/else for conditional branching, while for indefinite loops, for for definite loops, and match/case for structural pattern matching (Python 3.10+).

● Beginner 📖 Python Programming (2024)

1 · if / elif / else

Python uses indentation (4 spaces) instead of curly braces to delimit code blocks.

Python · if/elif/else 
# Basic if/elif/else
score = 78

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"Score {score} → Grade {grade}")  # Score 78 → Grade C

# Nested conditions
x, y = 10, 20
if x > 5:
    if y > 15:
        print("Both conditions met")

# Compound conditions
age  = 25
name = "Alice"
if age >= 18 and name != "":
    print("Valid adult user")

# Chained comparisons (unique to Python!)
n = 15
if 10 < n < 20:
    print("n is between 10 and 20")
▶ Output
Score 78 → Grade C
Both conditions met
Valid adult user
n is between 10 and 20

2 · Truthy & Falsy Values

Python determines truth through the bool() function. These values are all falsy:

Value Type bool()
False bool False
None NoneType False
0, 0.0, 0j numeric False
"", '' str False
[], (), {} containers False
set() set False
Everything else — True
Python · Truthiness 
# Pythonic: check if list is non-empty
items = [1, 2, 3]
if items:                      # Much better than: if len(items) > 0
    print("List has items")

# Check if string is non-empty
name = "Alice"
if name:
    print(f"Hello, {name}")

# or shorthand for default values
user_input = ""
value = user_input or "default"
print(value)   # "default" (because "" is falsy)
▶ Output
List has items
Hello, Alice
default

3 · Ternary Expression

Python's single-line conditional: value_if_true if condition else value_if_false

Python · Ternary Expressions 
# Classic ternary
age   = 20
label = "adult" if age >= 18 else "minor"
print(label)    # "adult"

# In function return
def abs_val(x):
    return x if x >= 0 else -x

# In comprehension
nums   = [1, 2, 3, 4, 5]
labels = ["odd" if n%2 else "even" for n in nums]
print(labels)   # ['odd', 'even', 'odd', 'even', 'odd']
▶ Output
adult
['odd', 'even', 'odd', 'even', 'odd']

4 · match / case (Python 3.10+)

Python's structural pattern matching — more powerful than a switch statement:

Python 3.10+ · match/case 
# Simple value matching
command = "quit"

match command:
    case "quit":
        print("Quitting...")
    case "hello":
        print("Hello!")
    case _:          # wildcard (default)
        print("Unknown command")

# Pattern matching with types
def describe(value):
    match value:
        case int() if value > 0:
            return "positive integer"
        case int():
            return "non-positive integer"
        case str():
            return f"string of length {len(value)}"
        case list():
            return f"list with {len(value)} elements"
        case _:
            return "unknown type"

print(describe(42))       # positive integer
print(describe("hi"))     # string of length 2
print(describe([1,2,3]))  # list with 3 elements
▶ Output
Quitting...
positive integer
string of length 2
list with 3 elements

5 · Short-Circuit Evaluation

Python's and and or operators don't always evaluate both sides. They stop as soon as the result is determined.

Python · Short-Circuit 
# and: stops at first False
# Returns: first falsy value, OR last value if all truthy
print(0 and 5)        # 0  (short-circuits at first falsy)
print(3 and 5)        # 5  (all truthy → returns last)

# or: stops at first True
# Returns: first truthy value, OR last value if all falsy
print(0 or 5)         # 5  (first is falsy → check next)
print(3 or 5)         # 3  (first is truthy → short-circuits)

# Practical use — safe default values
config = {}
host  = config.get("host") or "localhost"
print(host)   # "localhost" (because None is falsy)

# Safe attribute access
user = None
name = user and user.name    # Never calls user.name if user is None
print(name)   # None (safe!)
▶ Output
0
5
5
3
localhost
None
