# Python Learning Hub

A comprehensive, aesthetically rich learning website for Python, Data Science, and Machine Learning.

## 🚀 Features

- **63 Detailed HTML Pages**: Covering everything from Python basics to Advanced Deep Learning (Transformers, CNNs, etc.).
- **Interactive Design**: Modern CSS with glassmorphism, smooth gradients, and micro-animations.
- **Code Examples**: Practical, runnable Python examples for every topic.
- **Library Deep Dives**: Comprehensive coverage of NumPy, Pandas, Matplotlib, Seaborn, and Scikit-Learn.
- **Machine Learning & Deep Learning**: From-scratch implementations of Backpropagation, Attention mechanisms, and more.

## 📂 Project Structure

- `/pages`: Contains all the generated HTML content organized by topic.
- `/css`: Stylesheets for the website.
- `/js`: JavaScript for interactivity and the AI assistant.
- `gen_template.py`: Core script for generating consistent HTML pages.
- `expand_*.py`: Modular scripts used to generate specific sections of the site.

## 🛠️ How to Use

Simply open `index.html` in any modern web browser to start learning.
